# Roue de Noël (mobile)

## Contenu
- `index.html`
- `style.css`
- `script.js`
- `bg-mobile.png`

## Mise en ligne sur GitHub Pages
1. Crée un repo GitHub (public) et upload ces fichiers à la racine.
2. Va dans **Settings → Pages**
3. **Source** : `Deploy from a branch`
4. Branche : `main` — dossier : `/ (root)`
5. Ouvre l'URL GitHub Pages générée.

## Personnaliser
- Change le texte gagnant dans `script.js`
- Remplace `bg-mobile.png` par ton décor si tu veux (garde le même nom).
